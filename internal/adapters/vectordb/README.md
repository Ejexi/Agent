# adapters/vectordb/

Vector database adapter. Implements `ports.VectorDB`.

## Subdirectories

| Directory              | Description                                    |
| ---------------------- | ---------------------------------------------- |
| [pgvector/](pgvector/) | pgvector (PostgreSQL extension) implementation |

## Purpose

Stores and retrieves vector embeddings for RAG (Retrieval-Augmented Generation) and similarity search.
